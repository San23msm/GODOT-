# UI (User Interface)

## Descripción

Esta carpeta contiene los recursos, escenas y componentes relacionados con la Interfaz de Usuario (UI) del proyecto.

Su propósito es centralizar todos los elementos visuales que permiten la interacción entre el usuario y la aplicación o videojuego, manteniendo una estructura organizada, modular y fácil de mantener.

## Objetivos

* Separar la lógica de la interfaz de la lógica del juego.
* Facilitar la reutilización de componentes visuales.
* Mantener una arquitectura escalable para futuras funcionalidades.
* Mejorar la experiencia de usuario mediante interfaces claras y consistentes.

## Contenido

Dentro de esta carpeta podrán encontrarse elementos como:

* Menús principales.
* Pantallas de configuración.
* Interfaces de pausa.
* HUD (Heads-Up Display).
* Barras de vida, energía y experiencia.
* Ventanas emergentes.
* Inventarios.
* Sistemas de diálogo.
* Botones y controles personalizados.
* Elementos visuales reutilizables.

## Organización

Se recomienda organizar los recursos en subcarpetas según su función:

```text
UI/
├── MainMenu/
├── HUD/
├── Inventory/
├── Dialogs/
├── Settings/
├── Components/
└── Assets/
```

## Consideraciones

Todos los componentes de esta carpeta deben diseñarse siguiendo principios de modularidad y reutilización, permitiendo su integración en distintos proyectos y facilitando futuras ampliaciones del sistema de interfaz.
